# SpendDNA – Financial Fingerprint Analysis
📌 Overview

SpendDNA is an industry-focused data analysis project that transforms raw, messy transaction data into meaningful financial insights. It simulates how real fintech companies analyze user spending behavior to deliver actionable insights and personalized summaries.

📌 Objective

The goal of this project is to decode a user’s financial fingerprint by cleaning transaction data, identifying vendors, categorizing expenses, and analyzing spending patterns over time.
